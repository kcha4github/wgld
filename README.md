# 用語

## 座標変換

### モデル変換（DirectX系ではワールド変換）
被写体となるモデルがどの位置に存在しているか。

### ビュー変換
実際にカメラがどの位置にあるのか、どこを向いているかを定義。

### プロジェション変換
三次元空間のどの領域を表示するか。（例：横に幅広く？縦長？）
